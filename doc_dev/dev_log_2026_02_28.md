### Feature: Audio & Subtitle Track Selection ✅
- ใช้ `ffprobe` ตรวจหา audio/subtitle tracks ที่ฝังในไฟล์วิดีโอ
- สแกนไฟล์ subtitle ภายนอก (.srt, .vtt, .ass) ที่อยู่ข้างๆ วิดีโอ  
- แปลง SRT/ASS → WebVTT อัตโนมัติ (browser รองรับเฉพาะ VTT)
- ใช้ `ffmpeg` extract embedded subtitle เป็น WebVTT
- Web UI: ปุ่ม Audio/Subtitle ใน player header + **Modal Popup** กลางหน้าจอเพื่อเลือก track
- **Fix:** เปลี่ยนจาก Dropdown เป็น Modal เพื่อแก้ปัญหาการคลิกยากและปุ่มหลุดขอบหน้าจอ
- **New:** เพิ่มระบบ **APK Download** สำหรับติดตั้งลง Android TV ผ่าน Browser โดยไม่ต้องใชสาย USB
