# การรัน Tv2Cast Server

## วิธีการรันในเครื่องปัจจุบัน
1. เปิด Terminal และเข้าไปที่โฟลเดอร์ `server`:
   ```bash
   cd server
   ```
2. ติดตั้ง dependencies (ทำครั้งแรกครั้งเดียว):
   ```bash
   npm install
   ```
3. เริ่มการทำงานของ Server:
   ```bash
   npm start
   ```
   *หมายเหตุ: หากต้องการรันในโหมด Development (ให้รีสตาร์ทอัตโนมัติเมื่อแก้โค้ด) ให้ใช้ `npm run dev`*

---

## การนำไปรันที่เครื่องอื่น (เครื่องเซิร์ฟเวอร์หรือเครื่องคอมพิวเตอร์ลำยอง)

### 1. เตรียมความพร้อม (Prerequisites)
เครื่องปลายทางต้องติดตั้ง:
- **Node.js** (แนะนำเวอร์ชัน 16 ขึ้นไป)
- **FFmpeg** (จำเป็นสำหรับการ Transcoding วิดีโอเพื่อความเข้ากันได้กับ TV)

### 2. ขั้นตอนการติดตั้ง
1. คัดลอกโฟลเดอร์ `server` ทั้งหมดไปยังเครื่องใหม่ (ยกเว้นโฟลเดอร์ `node_modules` เพราะจะถูกติดตั้งใหม่)
2. เข้าไปในโฟลเดอร์ `server` ผ่าน Command Prompt หรือ Terminal
3. แก้ไขไฟล์ `.env` เพื่อกำหนดค่าใหม่ให้เหมาะสมกับเครื่องนั้น:
   - `PORT`: พอร์ตที่จะให้ Server รัน (ค่าเริ่มต้นคือ 3456)
   - `VIDEO_DIRS`: พาธของโฟลเดอร์วิดีโอในเครื่องนั้น (เช่น `C:\Movies` หรือ `/home/user/videos`)
4. ติดตั้ง dependencies:
   ```bash
   npm install
   ```

### 3. การรัน
ใช้คำสั่งเดิม:
```bash
npm start
```

### 4. ข้อแนะนำเพิ่มเติม (Pro-tips)
- **Firewall**: ตรวจสอบให้แน่ใจว่าได้เปิด Port `3456` (TCP) และ `3457` (UDP สำหรับการค้นหาอัตโนมัติ) ใน Firewall ของเครื่องนั้น
- **Background Service**: หากต้องการรันแบบถาวร (เปิดเครื่องแล้วรันเอง หรือรันเบื้องหลัง) แนะนำให้ใช้เครื่องมืออย่าง **PM2**:
  ```bash
  npm install -g pm2
  pm2 start server.js --name tv2cast-server
  pm2 save
  ```
